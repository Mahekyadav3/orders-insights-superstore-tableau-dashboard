# 🛒 Order Insights from Superstore — Sales Dashboard

An interactive business intelligence dashboard built with **Tableau Public**, analyzing order-level data from the Superstore dataset. This project highlights sales trends, category performance, and order insights across different regions and segments.

## 📌 Key Insights Covered

- 💰 **Sales & Profit KPIs** — Overall business performance at a glance
- 🗂️ **Category & Sub-Category Breakdown** — Top and bottom performing product groups
- 🌏 **Regional & Segment Analysis** — Which regions and customer segments drive the most revenue
- 📅 **Order Trends Over Time** — Monthly and yearly sales patterns
- 🚚 **Shipping Mode Impact** — How delivery type affects profit margins

## 🚀 How to Run Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/Mahekyadav3/order-insights-superstore.git
   ```
2. Open `index.html` in any web browser
3. The Tableau dashboard will load automatically (requires internet connection)
